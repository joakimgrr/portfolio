Customer management system

Tila contacted me as they needed a simple and easy to use customer management system that would work well in tablets.

The structure was pretty clear. The system would have editable clients, visits for each client and coupon codes. Coupon codes could be made manually or created automatically by the system.

The frontend was made with basic HTML, CSS and jQuery combination and it was communicating with a custom made PHP backend. All the data was stored in a MySQL database

Tools used

  • Photoshop
  • HTML
  • CSS
  • jQuery
  • PHP PDO
  • MySQL
