Odemenu

A friend of mine told me that it would be nice to have a website that combines the weekly menus of three different restaurants where he and his coworkers eat daily. I had some free time and wanted to do a project that would use Angular.js combined with Node.js.

A Node.js application with Cheerio runs on the server and it is used to scrape the menus from the restarants website. It then serves the information through a HTTP API made with Request.

Frontend is made with Angular.js and with a touch of jQuery.

Tools used

  • HTML
  • CSS
  • Angular.js
  • jQuery
  • Node.js
  • Cheerio
  • Request
